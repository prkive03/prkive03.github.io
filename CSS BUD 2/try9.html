COC 1 Installing and Configuring Computer Systems

This competency focuses on assembling computer hardware and installing operating systems.

Main tasks:
  • Install motherboard, CPU, RAM, storage devices
  • Connect power supply correctly
  • Install Windows or Linux OS
  • Install drivers
  • Configure BIOS settings
  • Install application software
Importance:

Ensures computer runs efficiently, safely, and correctly.

COC 2 Setting up Computer Networks

This competency focuses on connecting computers and configuring network systems.

  • Connect LAN cables
  • Configure IP address
  • Setup routers and switches
  • Test network connectivity
  • Configure internet access
Importance:

Allows computers to communicate and share resources.

COC 3 Setting up Computer Servers

Focuses on installing and managing server systems.

  • Install server operating system
  • Create user accounts
  • Manage permissions
  • Configure file sharing
  • Maintain server security
Importance:

Servers manage networks and provide services.
