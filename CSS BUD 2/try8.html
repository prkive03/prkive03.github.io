COC 1 Installing and Configuring Computer Systems

This includes assembling hardware components such as motherboard, CPU, RAM, power supply, storage devices, and peripherals. You must install operating systems like Windows or Linux, install drivers, configure BIOS settings, and ensure system stability. Proper installation ensures system performance, compatibility, and reliability.

COC 2 Setting up Computer Networks

This includes connecting LAN cables, configuring IP address, subnet mask, gateway, and DNS. Network devices include routers, switches, and access points. You must test connectivity using ping command and configure file sharing. Networking allows computers to communicate and share resources.

COC 3 Setting up Computer Servers

This includes installing server OS such as Windows Server or Linux Server. Configure user accounts, permissions, file sharing, and network services. Servers provide centralized management, security, and services for clients. Server maintenance ensures stability and security.
